Cloud

Was ist Cloud?

Ein Verbund von Computern, oder gemeinsam genutzen Rechnerressourcen.

Merkmale des Cloud Computing

On demand self-service

Benutzer haben jederzeit Zugriff und können selbst Dienste bestellen und verwalten.

Broad network access

Der Resource Pool ist über das Internet zugreifbar.

Resource Pooling

Viele Ressourcen werden an mehrere Kunden aufgeteilt.

Rapid elasticity

Ressourcen Anpassung - Ressourcen können nach Bedarf ausgebaut werden.

Measured service

Überwachung des Dienstes - Server werden ständig überwacht und optimiert. Hat mit Zahlung zu tun.

Dienstmodelle

Infrastructure as a service (Iaas)

Physische Server werden durch den Anbieter verwaltet und gewartet. Betriebssystem und Software werden selbst verwaltet.

Platform as a service (Paas)

Laufzeitumgebung für Applikationen wird durch Anbieter zur Verfügung gestellt. Applikation wird selbst verwaltet

Function as a service (Faas)

Laufzeitumgebung inkl. Middleware (z.B. HTTP Server) wird zur Verfügung gestellt. Nur die eigentliche Funktion selbst wird selbst verwaltet

>Software as a service (Saas)

Die gesamte Applikation wird durch den Anbieter zur Verfügung gestellt.

Deployment Modelle

Public

Benutzer hat keine Kontrolle über Infrastruktur. Jeder kann sich registrieren.

Private

Bereitstellung im eigenen Rechenzentrum. Man hat volle Kontrolle. Enorme Aufwand und Koasten.

Hybrid

Mischung von Private und Public. Beispiel: man kann die eigenen physischen Ressourcen mit ein open source Interface mischen.

Community

Mehrere Firmen scchliessen sich zusammen umgemeinsam ein Cloud zu betreiben.

The cloud is a metaphor for a global network of remote servers that operates as a single ecosystem, commonly associated with the Internet.

Overall, cloud computing has become integral to modern IT infrastructure, enabling businesses and individuals to leverage powerful computing resources without the need for extensive on-premises hardware and infrastructure.
